_A tool to compress quality values associated with next-generation genome sequencing data (C)_

<!--more-->

This [project](https://sourceforge.net/projects/qualcomp/) uses rate-distortion theory to provide an implementation of a lossy compression algorithm for the quality values in FASTQ files.  For more details, see the [paper]({% post_url 2013-06-08-qualcomp_quality_scores %}).
